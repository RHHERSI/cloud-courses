# AWS Snowball by Ryan Kroonenburg

## What is Snowball ?

Snowball is a petabyte-scale data transport solution that uses secure appliances to transfer large amount of data into and out of AWS.

Using Snowball addresses common challenges with large-scale data transfers including high network costs, long transfer times, and security concerns. Transferring data with Snowball is **simple, fast, secure and can be as little as one-fifth the cost of high-speed internet**

* 50 TB or 80 TB
* Multiple layers of security
* 256 bit encryption
* Trusted Platform Module (TPM)
* Ensure both security and full chain-of-custody of your data

### Snowball Edge

100TB data transfer device with on-board storage and compute capabilities. Can Use it to move large amounts of data into and out of AWS, as a temporary storage tier for large local datasets, or to support local workloads in remote or offline locations.

It connects to existing applications and infrastructure using standard storage interfaces, streamlining the data transfer process and minimizing setup and integration.

It can cluster together to form a local storage tier and process data **on-premises**.

### AWS Snowmobile

Exabyte-scale data transfer service used to move extremely large amounts of data to AWS.
Up to 100PB per Snowmobile

* 45-foot long shipping container pulled by a semi-trailer truck
* Move massive volumes of data to the cloud
* Secure, fast and cost effective
